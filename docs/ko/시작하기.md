## 준비물
- IBIGENT Studio 구독(활성, https://www.ibigent.com/service/ibigent-studio.html).
- MCP를 지원하는 AI 도구: Claude Code, Codex CLI, Cursor, Antigravity.

## 1단계 — MCP 키 발급
**마이페이지 > AI 연동 > 연동 키 관리** (https://www.ibigent.com/mypage/mcp-keys.html) 에서 **[키 발급]**. 1회 표시되는 `ibg_…` 값을 복사해 안전히 보관합니다.

## 2단계 — AI 도구 연결
아래 서버 주소를 등록(`<내_키>` 교체). 도구별 상세: [Claude Code](claude-code.md), [Codex CLI](codex.md), [Cursor](cursor.md), [Antigravity](antigravity.md).

```
https://mcp.ibigent.com/mcp/<내_키>
```

## 3단계 — AI로 제작
연결된 AI 도구에서 섹션 HTML을 만들어 IBIGENT Studio 편집 화면으로 보내도록 요청하고, 편집 화면에서 삽입 위치를 고릅니다. [예제](examples.md) 참고.
