# 설치방법 — IBIGENT Studio

**[문서 홈](../../README_ko.md)**

IBIGENT Studio는 **호스팅 서비스** 입니다 — 별도로 설치할 프로그램이 없습니다. “설치”는 구독하고 MCP 키를 발급받는 것을 뜻합니다.

## 요구사항
- IBIGENT Studio 구독(활성, https://www.ibigent.com/service/ibigent-studio.html).
- MCP를 지원하는 AI 도구: Claude Code, Codex CLI, Cursor, Antigravity.

## 설정
1. 공식 사이트에서 IBIGENT Studio 구독.
2. MCP 키 발급: **마이페이지 > AI 연동 > 연동 키 관리** (https://www.ibigent.com/mypage/mcp-keys.html).
3. 도구에 MCP 서버 등록 — [MCP](mcp.md) 참고.
