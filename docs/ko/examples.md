# 예제 — IBIGENT Studio

**[문서 홈](../../README_ko.md)**

AI 도구에 IBIGENT Studio MCP를 연결한 상태에서, AI에게 HTML 제작을 요청해 편집기를 움직입니다.

## 기본 흐름
1. IBIGENT Studio에서 편집 화면을 엽니다.
2. AI 도구에 섹션을 HTML로 만들고 IBIGENT Studio 편집 화면으로 보내달라고 요청합니다.
3. 편집 화면에서 삽입 위치를 고르면 그자리에 HTML이 들어갑니다.

## 예시 프롬프트
```
반응형 "서비스 소개" 섹션을 카드 4개로 HTML로 만들어서
내 IBIGENT Studio 편집 화면으로 보내줘.
```

AI가 HTML을 생성하고 MCP 연결을 통해 편집기로 전달합니다. 삽입 위치는 사용자가 고릅니다.
편집기 화면 상세 단계는 공식 문서 참고: https://www.ibigent.com/docs/ibigent-studio/
