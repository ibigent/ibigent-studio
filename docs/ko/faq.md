# FAQ — IBIGENT Studio

**[문서 홈](../../README_ko.md)**

### IBIGENT Studio란?
AI가 만든 HTML을 MCP로 받아 페이지 원하는 위치에 배치하는 호스팅형 실시간 웹 편집기입니다.

### 무언가 설치해야 하나요?
아니요. 호스팅 서비스입니다. 구독·MCP 키·MCP 지원 AI 도구가 필요합니다.

### 키는 어디서 받나요?
**마이페이지 > AI 연동 > 연동 키 관리**: https://www.ibigent.com/mypage/mcp-keys.html

### 키를 공유해도 되나요?
안 됩니다. 비밀번호처럼 취급하세요. 커밋·이슈 게시 금지, 노출 시 재발급.

### 어떤 AI 도구를 지원하나요?
Claude Code, Codex CLI, Cursor, Antigravity. VS Code는 준비 중입니다.

### VS Code에서 되나요?
공식 가이드 준비 중 — [vscode.md](vscode.md) 참고.

### 이 저장소에 제품 소스가 있나요?
아니요. 문서 전용입니다.
