# Installation — IBIGENT Studio

**[Docs home](../../README.md)**

IBIGENT Studio is a **hosted service** — there is no separate program to install. “Installation” means subscribing and issuing an MCP key.

## Requirements
- An active IBIGENT Studio subscription (https://www.ibigent.com/service/ibigent-studio.html).
- An MCP-capable AI tool: Claude Code, Codex CLI, Cursor, or Antigravity.

## Set up
1. Subscribe to IBIGENT Studio on the official site.
2. Issue your MCP key: **My Page > AI Integration > Connection Keys** (https://www.ibigent.com/mypage/mcp-keys.html).
3. Register the MCP server in your tool — see [MCP](mcp.md).
